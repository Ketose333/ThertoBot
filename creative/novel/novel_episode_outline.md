# 그저 깨어 있기를 바랄 뿐 — 에피소드 아웃라인

하위 리라이트 맵(1~15화): [`novel_episode_outline_ep1_ep15.md`](./novel_episode_outline_ep1_ep15.md)

## 전체 축
- 색 10개 회수 미션
- 승세 기억 복구 + 사건 진실 접근
- 코즈믹 침탈(검은 존재)과 서사 동화 대응
- 근미래 첨단 도시(외부) + 이상 구역(내부)의 이중 구조를 기본 무대로 유지

## 설정 동기화 기준
- 세계관 원문 기준: [`novel_story_bible.md`](./novel_story_bible.md)
- 1~15화 장면 적용 기준: [`novel_episode_outline_ep1_ep15.md`](./novel_episode_outline_ep1_ep15.md)

## 에피소드 1 (학교 루프 / 인트로)
- 포지션: 메인축이 아닌 데뷔 서사
- 핵심 키워드: 기억조작 / 팀 결성 / 기둥 피해자의 일상 고착
- 감정 결: 몽글한 추억 + 미세한 어긋남
- 적대 축: 검은 존재(서사를 배려하지 않고 분위기를 침탈)
- 종료 조건: 반주하 구출 + 색 회수
- 반주하 후속: 팀 합류 없이 단기 보호 후 일상 복귀
- 주의: 검은 존재 대응은 생존/차단 문제이고, 색 회수는 별도 조건으로 분리해 설계

## 초반 노출 규칙
- 대표-승세 관계: 3화 미세 암시, 6화 이후 본격
- 1~5화 직접 단서 노출: 최대 3회
- 호칭 규칙: 공식=대표님 / 사적=아저씨

## 주요 상태값
- 서사 동화: 배경 학생화 + 원래 역할 상실
- 동화 지연: 의식으로 늦출 수 있으나 공간 흔들림(검은 존재 침입) 시 취약
- 색 부작용:
  - 두통/감각 과민
  - 기억 플래시백
  - 감정 과부하(누적 색 증가 시 강화)

## 인물 반응 축
- 태율-진혁: 티격태격 협력(긴장공존)
- 대표-승세: 수상한 보호자-피보호자 톤
- 태율/진혁은 초반에 양부 관계 비인지
- 관계 공개 시: 태율은 비교적 덤덤, 진혁이 더 놀람

## 후속 에피소드 메모
- [ ] 색 2~3회수 구간에서 감정 과부하 상승을 단계적으로 표현(초기 감정축: 그리움)
- [ ] 타 팩션 협력 에피소드(외부 현실 파트 예외 확장) 배치 시점 확정
- [ ] 대표의 부분 공개/위계 유지 방식이 과잉 설명이 되지 않도록 장면화
- [ ] 3~5화 액션:정서 비율은 5:5 균형 유지
- [ ] 팀 5인 공개는 최대한 빠르게 하되, 정보 난잡화는 금지
- [ ] 소개문 톤은 서늘한 시적형 + 직설 미션형 혼합
- [ ] 15화 장기 플랜은 [`novel_episode_outline_ep1_ep15.md`](./novel_episode_outline_ep1_ep15.md) 기준으로만 갱신

## 추가 확정값 (2026-02-26)
- 검은 존재의 1순위 침습 방식: 공간을 덮어쓴다
- 3화 대표-승세 미세 암시 방식: 대사 + 행동 병행
- 5화 학교 이탈 메인 감정: 해방감
- 6화 첫 기둥 지역 톤: 미정 ("다음 컴백 콘셉트" 감각 기준으로 추후 결정)

## 모순/충돌 보정안 (고정 적용)
- 학교 인트로가 길어져도 1~3화에서 메인 미션 신호(색/기둥/팀 목적)를 최소 1회씩 노출한다.
- 용어는 선설명 금지: 사건 체감 → 용어 부여 → 짧은 설명 순서로만 제시한다.
- 검은 존재는 "격퇴"와 "색 회수"를 분리한다. (전투=생존, 색=공명/해석/선택)
- 기능 대사 과밀 방지: 한 장면에서 설정 설명 대사는 최대 2개 문장으로 제한한다.
- 톤 밸런스 규칙: 장면 단위로 추억/일상 70, 균열/침습 30 기본값을 유지한다.
- 팀 5인은 빠르게 소개하되, 한 화에 확정 인상 포인트는 최대 1인만 깊게 부각한다.
- 대표-승세 관계 단서는 1~5화 총 3회 이내, 3화 미세 암시·6화 이후 본격 원칙을 고정한다.
- 반주하는 구출 후 팀 합류시키지 않고 보호 후 일상 복귀시킨다.
